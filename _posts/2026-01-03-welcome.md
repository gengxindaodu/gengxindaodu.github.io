## What is Liquid?

Liquid is a templating language that lets you add dynamic content to your pages. Here are some examples:

### Variables
You can display variables using double curly braces:
- Page title: `{{ "{{ page.title " }}}}`
- Site title: `{{ "{{ site.title " }}}}`

### Loops
You can loop through collections:

```liquid
{{ "{% for post in site.posts " }}%}
  - {{ "{{ post.title " }}}}
{{ "{% endfor " }}%}
```

### Conditionals
You can use if/else statements:

```liquid
{{ "{% if site.posts.size > 0 " }}%}
  You have {{ "{{ site.posts.size " }}}} posts!
{{ "{% else " }}%}
  No posts yet.
{{ "{% endif " }}%}
```
